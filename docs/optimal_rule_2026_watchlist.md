## 🎯 발견된 최적 룰 2개

### Rule A — 가장 안전 (Highest Safety) ⭐⭐⭐

```
조건:
  - 직전 30일 종가 < $1.0
  - 거래량 평균 ≥ 10,000
  - 오늘 종가 in [$1.20, $1.50)
  - 어제 종가 < $1.20 (첫 돌파)
  - Warrant 제외 (W로 끝나는 종목)

진입: 다음날 시초가
청산: 진입가 +20% 도달 시 즉시 OR 90일 후 close
SL: 없음
```

**전체 검증 (2024.06 ~ 2026.06, 38건 완료):**
- 승률: **92.1%**
- 평균 수익: **+14.85%** (slippage 차감 후)
- 최악 (p10): **+18%** (모두 양수!)
- TP +20% 도달율: **92.1%**

### Rule B — 더 큰 수익

```
조건:
  - 직전 60일 종가 < $1.0
  - 오늘 종가 in [$1.10, $1.30)
  - Warrant 제외

청산: 진입가 +30% 도달 OR 180일 close
SL: 없음
```

**전체 검증 (43건 완료):**

## 💡 진짜 의미

### 새 룰의 강점 (Rule A 기준)
1. **승률 92% (in-sample) / 80% (2026 OOS)**: 이전 룰보다 압도적
2. **TP 도달 빠름 (median 1~6일)**: 자본 빠르게 회전
3. **작은 익절 + 자주 발생**: 심리적으로 지속 가능
4. **큰 손실 드물게 발생**: 1~3개 종목 (OLB, AMPGR 같은 케이스)

### 약점
1. **거래당 +14% 평균**: 이전 +79%보다 작음
2. **연 약 25건 시그널**: 자본 모두 활용하려면 분산 필수
3. **손실 종목은 −30~60% 가능**: SL 없으니 큰 폭락 감내 필요

### 2026년 in-sample → OOS 갭
- In-sample 검증 92% → 2026 실제 80%
- 12%p 하락 = **selection bias 또는 시장 환경 차이**
- 그래도 **80% 승률 + 평균 +14%**는 매우 강한 알파

## 🚨 정직한 한계

### 1. AMPGR / AMPGZ 같은 의심 종목
- **AMPG의 right (R) / unit (Z) 종목** = warrant-adjacent
- 일반 보통주 아닐 수 있음
- 자동매매에 부적합 (유동성 매우 낮음)
- **추가 필터 필요**: 종목 끝이 "R", "Z", "U" 제외

### 2. OLB 패턴 — TP 미도달 후 폭락
- 진입 $1.05 → 90일 후 $0.45 = −57%
- Rule A는 TP 도달 못 하면 **90일 close 청산 = 큰 손실 가능**
- 이전 룰 (180일)보다 빠른 청산이 양날의 검

### 3. 2026.05~06 시그널 결과 미확정
- AMPGR, AMPGZ, NTCL, TGHL 등 1~5월 시그널은 90일 안 됨
- 일부 TP 도달했지만 일부는 더 떨어질 수 있음

### 4. In-sample bias 잔존
- "win-rate 최대화" 그리드 검색 = overfitting 위험
- 1,800 조합 중 best 선택 = 우연 가능성
- 진짜 OOS 검증 (다른 시기 또는 paper trading) 필수
