## 🚨 정직한 한계

### 1. In-sample 그리드 search
각 윈도우 안에서 grid search → overfitting 가능.
다만 **다음 윈도우의 OOS 성과**가 평균 +13.67%이므로 일정 부분 신뢰 가능.

### 2. 13 윈도우는 작음
- 윈도우당 평균 7건 거래
- 통계적 신뢰도 보통
- 더 긴 기간 (10년+) 검증 필요

### 3. 1개 음수 윈도우 (26.01-03)
- 룰: 30d/$1.05-1.15/+10%/30d
- 결과: 9건 중 67% 성공이지만 평균 −12%
- **양수 보장 안 됨** — 분산 + 손실 한도 필수

### 4. Stooq 데이터 품질
- 가끔 결측 / 이상치 가능
- 실전 진입 시 yfinance 또는 Polygon으로 cross-check 권장

### 5. 슬리피지 2% 가정
- 페니스탁 실제 spread 5~10% 가능
- 실제 수익은 보고된 값보다 5~15% 낮을 가능성
- ₩18.7M → 현실 ~₩10M 정도 추정

### 6. 한국 거주자 양도세 22%
- ₩18.7M 수익 → 세후 ~₩14.6M
- 250만원 공제 적용

## 💡 진짜 의미

### 이전 정적 검증 → 단순 walk-forward → 적응형 walk-forward
각 단계마다 결과가 변화:

| 검증 단계 | 결과 |
|---|---|
| 1. In-sample 그리드 search | 승률 92%, +14.85% (환상) |
| 2. 단순 train/test split (1번) | 승률 68%, −0.5% (실망) |
| 3. **적응형 분기별 walk-forward** | **승률 83%, +13.67%** ✓ |

**진짜 답**: 시장이 변하므로 룰도 변해야 함.

### 핵심 통찰
- 페니스탁 시장은 **강세장 / 약세장 / meme 광풍**으로 환경이 바뀜
- 한 룰이 모든 환경에서 작동하지 않음
- **분기마다 직전 데이터로 재학습** = 시장 변화에 적응
- 적응형 시스템이 정적 룰의 ~13배 수익

### 그러나 100% 보장 안 됨
- 1개 윈도우 음수 (2026.01-03)
- 새로운 시장 환경에서 작동 보장 없음
- 분산 + 손실 한도 + 결과 모니터링 필수

## ⚠️ 면책

- **In-sample bias 가능성** 잔존
- **N=94는 통계적으로 작음**
- **2018-2022 등 다른 시장 환경 검증 안 됨**
- **슬리피지 / 세금 / 거래소 리스크** 미반영
- **반드시 paper trading 후 소액 실전부터**

과거 성과 ≠ 미래 보장.
