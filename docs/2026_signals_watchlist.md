## ⚠️ 실전 진입 전 고려사항

### 1. Warrant vs 일반 주식 분리

**권장**: Warrant 제외하고 일반 주식만 진입
- 유동성 + 데이터 품질 ↑
- 장기 평균 수익 더 좋음 (+89% vs +7%)

### 2. 데이터 신뢰도

- Stooq 데이터는 yfinance보다 약간 낮은 품질
- 일부 종목 split 처리 누락 가능
- 진입 전 yfinance로 cross-check 권장

### 3. 진입 가능 종목 (Warrant 제외, 현재까지 살아있음)

이미 돌파일이 지난 종목은 **추격 매수 위험**. 그러나 2026.06 기준 아직 TP 미달성하고 close가 진입가 근처인 종목:
- **TEAD**, **ABLV**, **WETO** (close 진입가 근처, +5% 이내)
- 단 이미 1~3개월 지났으니 catalyst 약화 가능성

### 4. 새 시그널 모니터링

매일 스캔 권장:
```python
# 매일 장 마감 후 실행
# 60d cons + 오늘 close $1.05~$1.20 돌파 종목 검색

```

자동화하면 새 시그널 발생 시 즉시 paper trading 시작 가능.
